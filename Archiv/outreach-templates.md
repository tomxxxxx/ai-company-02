# LinkedIn Cold Outreach Templates

## Template 1: Problem-Focused (Empfohlen)

**Betreff/Erste Nachricht:**
```
Hallo [Name],

ich habe gesehen, dass [Firma] auf LinkedIn aktiv ist. Eine Frage: Wie viel Zeit geht bei Ihnen pro Woche für Social Media Content drauf?

Die meisten Unternehmen sagen mir 15-20h. Wir haben ein KI-System entwickelt, das das auf unter 1h reduziert - ohne Qualitätsverlust.

Interesse an einem kurzen Austausch (15 Min)?

Beste Grüße,
Thomas
```

## Template 2: Value-First

**Betreff/Erste Nachricht:**
```
Hi [Name],

ich arbeite mit Unternehmen wie [Firma], um deren Content-Produktion zu automatisieren.

Quick Win für Sie: Schicken Sie mir 3 Ihrer besten LinkedIn-Posts, ich zeige Ihnen in 5 Min, wie KI das für Sie skalieren kann.

Kostenlos, keine Verpflichtung.

LG Thomas
```

## Template 3: Direct (für Entscheider)

**Betreff/Erste Nachricht:**

## Follow-Up Template (nach 3 Tagen keine Antwort)

```
Hi [Name],

verstehe, wenn gerade kein Fokus auf dem Thema liegt.

Falls doch interessant: Hier ein kurzes 2-Min Video wie das System funktioniert: [Link]

Schöne Woche!
Thomas
```

## Zielgruppen-Filter LinkedIn

**Suche nach:**
- Geschäftsführer / CEO / Founder
- Marketing Manager / Head of Marketing
- Unternehmen 5-50 Mitarbeiter
- Location: DACH-Region
- Aktiv auf LinkedIn (mindestens 1 Post pro Woche)

**Industries:**
- Professional Services
- Consulting
- Real Estate
- E-Commerce
- Coaching & Training
- SaaS

**Ausschlusskriterien:**
- Zu groß (>100 MA) = lange Sales Cycles
- Zu klein (<3 MA) = kein Budget
- B2C ohne Social Media = passen nicht
